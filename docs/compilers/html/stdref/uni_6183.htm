uninitialized_fill_n

     Memory Management

Summary

Algorithm that uses the language feature placement new for setting values in a collection.

Synopsis

 
#include <memory>
template <class ForwardIterator, 
          class Size, class T>
void uninitialized_fill_n (ForwardIterator first,
                           Size n, const T& x);

Description

unitialized_fill_n starts at the iterator first and initializes the first n items to the value x, using the language feature placement new.
