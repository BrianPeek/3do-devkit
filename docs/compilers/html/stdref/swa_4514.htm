swap

     Algorithm

Summary

Exchange values.

Synopsis

 
#include <algorithm>
template <class T>
 void swap (T& a, T& b);

Description

The swap algorithm exchanges the values of a and b. The effect is:

T tmp = a
a = b
b = tmp
